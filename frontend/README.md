## Getting Started

### Prerequisites

- Node.js 18+
- Xverse Bitcoin wallet
- MIDL Bitcoin testnet funds

### Installation

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build & Deploy

```bash
npm run build
firebase deploy
```

## How It Works

1. **Connect Wallet** - Link your Xverse wallet
2. **Browse Influencers** - Explore crypto leaders
3. **Choose Tier** - Select Chat, Call, or Coffee access
4. **Buy Access** - Send BTC, get instant access recorded on blockchain
5. **Access Unlocked** - View exclusive content

## Access Tiers

- **Chat** - Direct message access (30 days)
- **Call** - 30-minute strategy call
- **Coffee** - In-person meeting
